NASM

O Netwide Assembler (NASM) é um assembler e disassembler para a arquitetura Intel x86. Ele pode ser usado para escrever programas de 16-bit, 32-bit (IA-32) e 64-bit (x86-64). NASM pode produzir diversos formatos binários como COFF, OMF, a.out, Executable and Linkable Format (ELF), Mach-O e binary file (.bin, usado para compilar sistemas operacionais). NASM também possui seu próprio formato binário chamado RDOFF.

    Exemplo de uso: Hello, world!

Para criar e executar um executável a partir do trecho de código assembly abaixo salvo como “hello.asm” que exibe a mensagem “Hello, world!” no Linux, é necessário executar primeiramente o comando “nasm -f elf hello.asm” que irá gerar um object file (arquivo .o) para o programa. Depois é necessário usar um linker para traduzir o object file em um executável com o comando “ld hello.o -o hello”. Por fim, para executá-lo basta o comando “./hello”.

global _start

section .text
_start:
  mov eax, 4 ; write
  mov ebx, 1 ; stdout
  mov ecx, msg
  mov edx, msg.len
  int 0x80 ; write(stdout, msg, strlen(msg));

  mov eax, 1 ; exit
  mov ebx, 0
  int 0x80 ; exit(0)

section .data
msg: db "Hello, world!", 10
.len: equ $ - msg

KDBG

O KDBG é uma interface gráfica para o gdb, o debugger da GNU. Para debugar um programa, escolha um arquivo/carregue um executável no menu. É possível definir breakpoints com o botão direito do mouse ou na janela de breakpoints. Você pode executar o programa com o botão “Run”. O programa vai executar até terminar, atingir um breakpoint/watchpoint, ou receber um sinal. Também é possível executar o programa com argumentos, escolher o diretório de trabalho, e definir variáveis de ambiente. É necessário fazer isto antes de iniciar o programa.

Bless Editor

O Bless Editor é um editor hexadecimal que permite escrever, visualizar e editar arquivos em hexadecimal e ASCII simultaneamente. Ele também exibe a linha do texto atual em decimal, octal e binário. Também é possível visualizar no aplicativo cada byte em signed e unsigned em 8 bit, 16 bit, 32 bit, além de float 32 bit e 64 bit. O programa também permite copy/paste e busca no arquivo em todos os formatos de representação de dados citados anteriormente.
